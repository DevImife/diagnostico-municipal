# 📚 Sistema de Captura de Información de Escuelas

Sistema web desarrollado para la captura, administración y consulta de información detallada de planteles educativos.

Permite registrar datos generales, infraestructura, condiciones físicas, espacios educativos y características constructivas de cada escuela.

---

## 🎯 Objetivo del sistema

Centralizar la información técnica de las escuelas para:

- Evaluación de infraestructura educativa
- Planeación de mantenimiento y rehabilitación
- Generación de reportes institucionales
- Seguimiento histórico de condiciones físicas
- Toma de decisiones administrativas

---

## 🧩 Módulos principales

- Registro de escuelas
- Formulario extenso de infraestructura
- Registro de edificios
- Registro de espacios educativos
- Condiciones físicas y materiales
- Consulta y edición de información
- Reportes

## 📝 Descripción del formulario principal

El sistema cuenta con un formulario dividido en secciones:

- Datos generales del plantel
- Ubicación
- Información del terreno
- Número de edificios
- Características estructurales
- Materiales de construcción
- Estado físico
- Servicios (agua, luz, drenaje)
- Observaciones

Cada escuela puede tener múltiples edificios y cada edificio múltiples espacios.
